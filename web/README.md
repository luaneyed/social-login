# Social Login Client

This app assumes that the server is running on `http://localhost:9000`

## What is used?
- React.js 17.0
- TypeScript 4.2.4
- Webpack 5.36.2
- Sass

## Preparations
- Install Node.js 14
- Install Yarn
- Run `yarn install`

## How to Run
```
$ yarn run start-prod
```
The app is served at `http://localhost:8080`
